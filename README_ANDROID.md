# VP Tennis Academy - Android Source Code

This folder contains the complete, pre-configured Android project for your app. It has already been synced with Capacitor and contains all your web assets.

## How to build your APK from this source:

You will need a computer (Windows, Mac, or Linux) that has **Android Studio** installed.

1. **Copy** this entire extracted folder to the computer with Android Studio.
2. **Open Android Studio**.
3. On the welcome screen, click **Open** (or File > Open).
4. Select the `android` folder located right next to this README file.
5. Wait for the project to fully load. Watch the bottom progress bar that says "Gradle: Syncing...". Wait until it completely finishes and says "Gradle sync finished successfully".
6. In the top menu bar, click **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
7. Android Studio will compile the app. When it's done, a small popup will appear in the bottom right corner with a link saying "locate". 
8. Click **locate** to open the folder containing your final `app-debug.apk` file.

You can now copy that `.apk` file to your Android phone and install it!
